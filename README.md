## 📌 Project Overview
This project is a fully responsive one-page e-commerce landing page built using Wix. It demonstrates dynamic product management using CMS, custom functionality using Velo, and real-time order tracking using Google Sheets integration.

---

## 🚀 Features
- Responsive one-page e-commerce design  
- CMS-driven dynamic product management  
- Product search and filtering functionality  
- Add to Cart functionality using Velo  
- Wix Stores checkout integration  
- Additional Fees applied via backend logic  
- Google Sheets integration for order tracking  
- SEO optimized structure and performance  

---

## 🛠️ Tech Stack
- Wix  
- Velo (JavaScript)  
- Wix CMS  
- Google Sheets Integration
